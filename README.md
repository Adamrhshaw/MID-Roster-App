# Adam Rostering

Staff rostering app for a NSW Health radiology department. Managers build and manage 4-week roster blocks across X-Ray, Ultrasound, and CT — assigning morning, afternoon, and night shifts while the rules engine flags rest-period, hours, leave, and certification violations. Staff submit leave requests and swap proposals through a separate no-auth portal.

**Stack:** Next.js 16 · Supabase (PostgreSQL + Auth) · shadcn/ui v4 · Tailwind CSS v4 · TypeScript
